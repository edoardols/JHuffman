
Una volta eseguito il file HuffmanCodecmd.jar richiederà di inserire un comando

ENCODE : comincia la parte di codifica (richiesta di inserire un testo e successivamente mostrerà i risultati della codifica, della decodifica,
	la tabella delle frequenze e il codice di Huffman per ogni tipo di carattere presente nel testo inserito, e le statistiche dell'algoritmo (tempi e fattore di compressione)

EXIT: si uscirà dall'esecuzione

NB: alla fine dell'esecuzione del comando ENCODE il programma richiederà se si vuole codificare un altro file o uscire dall'esecuzione
